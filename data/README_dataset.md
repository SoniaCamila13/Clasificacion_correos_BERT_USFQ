# Dataset

El dataset utilizado en este proyecto contiene correos electrónicos institucionales anonimizados pertenecientes al área de Registro de la Universidad San Francisco de Quito (USFQ).

Por motivos de privacidad y confidencialidad institucional, los datos originales no se encuentran disponibles públicamente.

El dataset fue sometido a procesos de:

- Limpieza y depuración
- Eliminación de ruido
- Filtrado de contenido irrelevante
- Anonimización mediante técnicas NER utilizando SpaCy
