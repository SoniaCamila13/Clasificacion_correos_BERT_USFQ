
# Clasificación de Correos con BERT

Proyecto de titulación enfocado en la automatización de la clasificación de correos electrónicos mediante NLP.

## Descripción
Se implementa un modelo BERT para clasificar correos en categorías administrativas.

## Tecnologías
- Python
- Transformers (Hugging Face)
- Scikit-learn
- Google Colab

## Resultados
F1-score: 0.96
